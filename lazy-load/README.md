# Pre- or lazy- loading audio, image or video assets

- Using the HTML `preload` attribute for [audio](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/audio#preload) or
[video](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/video#preload) assets, or 
the `loading` attribute for [`img`](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/img#loading) assets, 
allows easy no-JavaScript pre- or lazy-loading those assets.
