# Input with Autofilter Suggestions Dropdown

aka: Combobox

- Using the HTML [`datalist`](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/datalist) and [`input`](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/input) elements creates no-JavaScript dropdowns that autofilter as you type.
- Similar to old-school "comboboxes", curated list in dropdown autofilters as you type.
- But you can also add a custom entry.
