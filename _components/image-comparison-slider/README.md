# Image Comparison Slider

aka: Side-by-Side Comparison

- Using CSS [`anchor-name`](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/anchor-name) and
            [`anchor()` function](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Values/anchor) creates a no-JavaScript 
            image comparison tool.

## [Basic](basic)
Compare two images, one over the other, by panning a slider bar back and forth.

## [Custom Handle](custom-handle)
Same as the [Basic](basic), but with some UI customization.
