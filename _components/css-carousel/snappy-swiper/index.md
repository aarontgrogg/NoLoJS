# CSS Carousel - Snappy Swiper
With one line of additional CSS, the slides "snap" into place as the user swipes (in the center, in this case). 
Note that even when dragging the scrollbar, the "snap to the center" still happens after releasing the scrollbar, all CSS, no JS required.
